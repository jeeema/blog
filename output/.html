date:2017-10-14 10:55:50

Inline markers

The right-hand forms are called "interpreted text roles". The semantics for this is like this; <:rolename:`text`>

reST code:

*emphasis* = :emphasis:`emphasis`

**strong emphasis**  = :strong:`strong emphasis`

``literal``  = :literal:`literal`

:superscript:`superscript` :subscript:`subscript`

This is printed like this:

emphasis = emphasis

strong emphasis = strong emphasis

literal = literal

superscript subscript

Lists

Normal lists(including the numbered and the non-numbered)

reST code:

* This is
* a bulleted list.

1. AAAAA

     #. aaaaa

         #. xxxxx


     #. bbbbb


#. BBBBB
#. CCCCC
#. DDDDD
#. EEEEE

This is printed like this:

  • This is
  • a bulleted list.

  1. AAAAA

    1. aaaaa

      1. xxxxx

    2. bbbbb

  2. BBBBB

  3. CCCCC

  4. DDDDD

  5. EEEEE

  1. vvvvv
  2. wwwww
  1. xxxxx
  2. yyyyy

Definition lists

reST code:

term1
    description of term1
term2
    description of term2

    This line is still the part of discription of term2.



term3
    description of term3

This is printed like this:

term1
description of term1
term2

description of term2

This line is still in the discription of term2.

term3
description of term3

We can see 3 blanks lines between term2 and 3 are removed. It seems the blank lines between the entries(?) are removed.

Literal blocks

The blocks after literal block markers will be unindented, and the font of each block will be a monospaced font, in all cases. There must be more than one line between a line containing the marker and the literal block.

If the markers occur following a non-space character, single colon following that non-space character will be printed:

The next paragraph is a code sample::

    def func():
        print("Hello World")

So this is printed like this:

The next paragraph is a code sample:

def func():
    print("Hello World")

If the markers occur following a space, the colons are not printed:

The next paragraph is a code sample. ::

    def func():
        print("Hello World")

So this is printed like this:

The next paragraph is a code sample.

def func():
    print("Hello World")
